MetaQuerySpec¶

public interface MetaQuerySpec<T extends QuerySpec>¶

MetaQuerySpec provides implementations of metadata-related methods related to a given QuerySpec.

Instances of this interface are primarily used internally and instantiating it is expensive; if you need it, an instance is available via ParsedQuery.getMetaData()

Author:tdavis

getFilterTable¶

Table<BoundFilterEntry<T>, String, BeanPropertyDefinition> getFilterTable()¶

Returns a table of filters where rows are their internal representation, columns are a given string alias for the filter, and the value is the meta description of the annotated field.

getFiltersForField¶

Class<? extends Filter>[] getFiltersForField(String name)¶

tableFor¶

Table<BoundFilterEntry<T>, String, BeanPropertyDefinition> tableFor(BeanPropertyDefinition field, Class<? extends Filter>... filters)¶

Provides a narrowed view of the table provided by getFilterTable() by looking at only a specific field and set of filters.

This method should not perform validation on the entries in filters to, e.g., check for existence of the filter on the field.
