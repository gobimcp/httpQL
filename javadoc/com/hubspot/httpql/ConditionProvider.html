ConditionProvider¶

public abstract class ConditionProvider<T>¶

Used by Filter implementations to create appropriately-typed Condition instances.

Because Filters are (necessarily) un-typed and Field and Param instances must share the same type parameter, this class acts as a closure of sorts to make sure that’s possible.
