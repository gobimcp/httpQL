QueryParser¶

public class QueryParser<T extends QuerySpec>¶

Primary entry point into httpQL.

The parser’s job is to take a set of query arguments (string key/value pairs) and turn it into a high-level query representation, assuming it is valid according to the defined filtering rules (provided via FilterBy and OrderBy annotations).
